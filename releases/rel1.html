Release Notes 1.0 - Feb 2016¶

This is the first production release of MCViNE.

Installation¶

The release can be installed on ubuntu 14.04 and fedora 21 using debian and rpm package managers, respectively.

Publications¶

The introduction to the architecture and design principles and some key algorithms for the MCViNE software framework was published at Nuclear Instruments and Methods in Physics Research Section A.

More about publications can be found at Publications.

Interface¶

The main command line interface (CLI) was established in the mcvine CLI, in which a series of simulation applications for the ARCS spectrometer are included.

Source code¶

can be found at https://github.com/mcvine/mcvine/releases/tag/1.0.
