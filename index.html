MCViNE: Monte Carlo VItual Neutron Experiment¶

MCViNE is a versatile Monte Carlo neutron ray-tracing program that provides researchers with tools for performing computer modeling and simulations that mirror real neutron scattering experiments.

Currently MCViNE software is only available for the Linux platform. Detailed installation instructions for MCViNE are available at Installation. Building the source is driven by a python script, but the underlying build engine is based on gnu make.

The MCViNE user documentation includes explanations of MCViNE concepts, tutorials, usage of neutron components, and instructions for creating a sample assembly.

At this point, MCViNE is under the git version control system, and MCViNE source code is available for online-browsing at the github site.

The MCViNE software is an open source software and is freely available. More details about the conditions of use and license can be found here. Feedback to the MCViNE developers can be provided through the MCViNE user mailing list.
